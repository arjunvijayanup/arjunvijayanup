## 🚀 Featured Projects

### ♟️ [Chess DNN + Alpha-Beta Engine](https://github.com/arjunvijayanup/chess-dnn-alpha-beta)
> *Hybrid chess engine combining search algorithms with deep learning*

A chess engine that merges **negamax α–β search** with a batched **DNN evaluator** using 782-D NNUE-style features. Includes transposition tables, killer/history move ordering, an opening book, and Stockfish benchmarking.

`Python` · `TensorFlow` · `python-chess` · `Alpha-Beta Pruning` · `Deep Learning`
